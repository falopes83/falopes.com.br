UX Case -
Controle de Tarefas

Websupply UX/UI 2024

Sobre o Projeto

O Controle de Tarefas é um display desenvolvido para apresentar as equipes e seus respectivos colaboradores de forma clara e eficiente. Cada card no display representa um colaborador e exibe informações essenciais como a tarefa atual, o tempo dedicado a ela, as horas trabalhadas no dia, o cliente associado e o status no Teams. Este último detalhe é importante, pois indica se o colaborador está ausente ou em call com outro membro da equipe.

O Problema

O sistema anterior utilizava um modelo de tabela, que se tornou cada vez mais difícil de manejar conforme o número de colaboradores crescia. A visualização das informações era confusa, e localizar um colaborador específico na tabela se tornou um desafio.

Objetivos do Projeto

O novo display foi criado com os seguintes objetivos:

  • Substituir o Modelo de Tabela por Cards: Adotar um layout de cards para uma visualização mais intuitiva e organizada, facilitando a identificação dos colaboradores e suas respectivas tarefas.
  • Implementar um Layout Dark: Escolher um design dark para oferecer um visual mais moderno e atraente, garantindo uma leitura mais confortável em ambientes de trabalho diversos.
  • Melhorar a Eficiência na Localização de Informações: Simplificar o processo de localização e compreensão das informações, tornando o display mais funcional e acessível para os usuários.
